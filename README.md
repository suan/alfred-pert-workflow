# Alfred P.E.R.T. Calculator Workflow
Calculate [P.E.R.T.][pert_wikipedia] estimates right in Alfred!

## Usage (in Alfred)

    pert <optimistic> <most likely> <pessimistic>

Example:

    pert 2 3 5

If \<Enter\> is pressed the output is copied to your clipboard.
